# Java Memory Management

This module explains how Java manages memory at runtime using the JVM.

## Topics Covered
- Stack vs Heap memory
- Object lifecycle and garbage collection
- Reference types (Strong, Weak, Soft, Phantom)
- OutOfMemoryError scenarios
- JVM memory behavior

## Key JVM Areas
- Stack (Thread-specific)
- Heap (Shared)
- Metaspace
- Program Counter
- Native Method Stack

```bash
javac *.java
java HeapStructure
