# Java Design Patterns

This module demonstrates **enterprise-grade Java Design Patterns**
implemented using **real backend use cases** inspired by
food delivery and large-scale systems.

## Patterns Covered

### Creational
- Singleton (Thread-safe)
- Factory Method
- Builder

### Structural
- Adapter
- Decorator
- Proxy

### Behavioral
- Strategy
- Observer
- Command

## Backend Relevance
- Payment gateways
- Order lifecycle management
- Authentication & authorization
- Feature extension without modification
- Clean separation of concerns
