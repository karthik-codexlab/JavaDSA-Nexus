# Java I/O and NIO Examples

This module demonstrates Java Input/Output and New I/O (NIO) concepts
used in real-world backend and system-level applications.

## Concepts Covered
- FileReader & FileWriter
- BufferedReader & BufferedWriter
- Serialization & Deserialization
- Path and Files API
- Java NIO file operations

## Why This Matters
These APIs are widely used in:
- Backend services
- Logging systems
- File processing jobs
- Data pipelines

This module strengthens core Java foundations required for
Spring Boot, Microservices, and System Design.
