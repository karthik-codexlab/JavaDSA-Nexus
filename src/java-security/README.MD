# Java Security & Cryptography

This module demonstrates core security concepts implemented using
Java's standard cryptography and security APIs.

## Topics Covered
- Hashing (SHA-256)
- Symmetric Encryption (AES)
- Digital Signatures (RSA)
- Secure Random Number Generation
- TLS / SSL Concepts

## Why This Matters
Security is fundamental for backend, fintech, defense, and distributed systems.
This module focuses on practical, production-relevant Java security usage.
